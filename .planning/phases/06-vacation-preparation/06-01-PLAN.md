Create `docs/reservoir-test-protocol.md` -- a one-page protocol for the 5-day reservoir dry-run test.

**Structure (follow established doc format from docs/):**

1. **Overview** -- What the test proves (reservoir sustains plants 5+ days without manual watering). Why we test early (by W26, leaving 2-3 weeks before departure for fallback planning).

2. **Which beds to test** -- Write generically: "For each bed with an installed reservoir." Note the bed identity discrepancy from RESEARCH.md (CLAUDE.md says A and B; data files say bed-c). Instruct: "Confirm which physical beds have reservoirs before starting the test."

3. **Test procedure:**
   - Fill reservoir completely via fill tube. Note the time.
   - Stop ALL manual watering for this bed.
   - Each morning for 5 consecutive days, child does the finger test:
     - Push finger into soil at bed edge to 5-10cm depth
     - Record: surface (dry/damp), root depth (dry/damp), notes
   - Father logs results in the daily log grid
   - Pass criteria: root depth stays damp all 5 days. Surface drying (top 2-3cm) is normal and acceptable.
   - Fail criteria: root depth goes dry on any day before Day 5.

4. **Daily log template** -- Table with columns: Day (1-5), Date, Soil Surface (Dry/Damp), Root Depth 5-10cm (Dry/Damp), Notes, Tester. Use the exact template from RESEARCH.md code examples section.

5. **If test fails** -- Increase neighbor visit frequency to every 2-3 days for failed beds. No new hardware (per locked decision). Note this in the neighbor guide.

6. **Child involvement** -- The child is the daily soil check partner. He does the finger test each morning and reports "wet" or "dry." This builds understanding of why the reservoir matters and why they are preparing before vacation.

**Formatting:** Keep it to one printable page equivalent. No fluff. Use the reservoir-build.md capacity specs (15-20L, 2-4 day autonomy) for context but do not duplicate the build instructions.

3. **Volume calculations** -- Use the exact table from RESEARCH.md:
   | Bed | Dimensions | Area | Volume at 5cm | Volume at 8cm | Material |

4. **Per-bed instructions** -- For each bed (A through E):
   - Material type
   - Depth target: 5-8cm
   - Special notes (e.g., keep mulch 2-3cm away from plant stems to prevent rot; terrace beds may need slightly thicker layer due to wind exposure)
   - Use a depth-check stick marked at 5cm and 8cm

5. **Child activity framing** -- "Tucking the plants in for their nap while we're away" (per locked decision). Mulch spreading is a sensory-rich activity (texture, smell). Child helps with all beds.

6. **Post-mulching** -- Deep water all beds immediately after mulching to settle the material and start moisture retention.

**Formatting:** Practical reference doc. Include the volume table for shopping. Keep concise -- this is a guide, not an essay.
