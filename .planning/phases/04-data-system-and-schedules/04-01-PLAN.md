  <action>
Create directory structure: `data/schemas/`, `data/crops/`, `data/beds/`.

Create `data/schemas/crop.schema.json` using JSON Schema Draft 2020-12. Required fields from CONTEXT.md locked decisions:
- `id` (string, lowercase-hyphen format)
- `name` object: `common`, `botanical`, `da` (Danish), `shortcode`
- `type` enum: fruit, vegetable, herb, flower, sensory
- `difficulty` object: `tier` enum (cant_fail, needs_care, year_2_challenge), `label` (en/da), `reason`
- `growth_stages` array of objects: `name`, `weeks` (start/end), `look_for`, `child_actions` array
- `child_actions` within growth_stages: `action`, `type` (harvest/observe/tend/sensory), `who` (child/father/together), `minutes`, `frequency` (daily/weekly/once), `prompt` (en/da bilingual)
- `water_needs`: `human_readable` (en/da), `thresholds` object (soil_moisture_min, soil_moisture_max, drought_alert_days)
- `temperature`: `optimal_min`, `optimal_max`, `frost_sensitive`, `frost_threshold`, `heat_stress`
- `harvest`: `first_harvest_week`, `last_harvest_week`, `days_to_maturity`, `continuous` boolean, `notes`
- `companion_plants`: `good` array, `bad` array, `notes`
- `engagement`: `action_types` array, `peak_engagement_weeks` (start/end), `attention_minutes_per_week`, `sensory` array
- `beds` array of bed IDs
- `sowing`: `method` (seed/plug/bare-root/division), `indoor_start` week or null, `outdoor_sow` week or null, `transplant_week` or null, `succession` boolean, `interval_weeks` (if succession)
- `alerts`: array of alert threshold objects with `type`, `trigger`, `threshold`, `delay_hours`, `message` (en/da)

Create `data/schemas/bed.schema.json` with required fields:
- `id`, `name`, `physical` (position, dimensions, material, features, orientation)
- `crops` array: `crop_id`, `shortcode`, `positions` array (center_w, center_l, spread_cm), `planted_week`
- `succession_slots` array: `id`, `area` (w_start, w_end, l_start, l_end), `crop_id`, `first_sow_week`, `interval_weeks`
- `ha_sensors`: `moisture`, `temperature` entity IDs

Create 3 template crop files covering the 3 difficulty tiers as reference patterns:
1. `strawberry-ostara.json` (cant_fail) -- use the exact example from 04-RESEARCH.md Pattern 1, add alerts field
2. `radish.json` (cant_fail, succession crop) -- 25 days to maturity, sow W16+, succession every 2 weeks
3. `cherry-tomato-sungold.json` (needs_care) -- frost-sensitive, transplant W21, growth stages from seedling through fruiting

Use the existing planting grid maps (docs/planting-grid-bed-*.md) for bed assignments, shortcodes, and sow weeks. Use Danish zone 7b timeline from research for harvest weeks and growth stage windows.

All IDs must use lowercase-hyphen format (e.g., `strawberry-ostara`, `bed-a`).
